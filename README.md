# menu-makanan-nusantara
Aplikasi ini menampilkan daftar menu makanan lokal yang terbagi dalam tiga kategori — Makanan Utama, Minuman, dan Camilan — dengan fitur filter kategori, rating bintang, dan harga dalam format Rupiah. Pengguna dapat menambahkan menu ke keranjang belanja, melihat detail tiap menu (deskripsi, kalori, bahan-bahan), serta menyimpan menu favorit.
